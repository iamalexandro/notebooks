• Arrays

• Sintaxis

let names = ['nicola', 'duban', 'jesus', 'edward']

• Acceder a una posicion

names [3] output: "edward "

• Desestructuracion

Iguala las posiciones del array a variables nuevas

let [s1,s2,s3,s4] = names

• Agregar y quitar elementos

• push

agregar a un array en la ultima posicion por defecto
let nunbers [1,2,3,4]
numbers.push(5) -> [1,2,3,4,5]

• pop

quitar del array el ultimo elemento

let nunbers [1,2,3,4]
numbers.pop() -> [1,2,3]

• unshift

agregar elemento al inicio del array

let nunbers [1,2,3,4]
numbers.unshift(5) -> [5,1,2,3,4]

• shift

elimina el primer valor del array y retorna el valor eliminado

let nunbers [1,2,3,4]
numbers.shift() -> [2,3,4]

• splice

agregar o eliminar en una posicion personalizada

.splice(startIndex, quantityDelete, value1, value2, value3)

Agregar dos elementos en el index [2]
let nunbers [1,2,3,4]
numbers.splice(2,0,9,9) -> [1,2,9,9,3,4]
agregar el numero 7 en el index [1] eliminando dos elementos
let nunbers [1,2,3,4]
numbers.splice(1,2,7) -> [1,7,4]

• slice

.slice(start,end) extrae un nuevo array los valores desde start hasta end -1. (NO MODIFICA EL ARRAY ORIGINAL)

let nunbers [1,2,3,4,5,6,7]
numbers.slice(2,5) -> 3,4,5

• Ordenar elementos de un array

• reverse

• ordena inversamente el array
'hola mama'.split('').reverse() -> ('a' 'm' 'a' 'm' '' 'a' 'l' 'o' 'h' )

• unir un array
let array = ['h','o','l','a'] | array.join('') -> 'hola'

• recibir un string invertirlo y unirlo
const reverseText = string => string.split('').reverse().join('')
reverseText('nicola') -> 'alocin'

• sort (strings)

ordena un array (solo strings)
let letters = ['c','d','a','b']
letters.sort() -> ["a", "b", "c", "d"]

• sort (callback) for numbers

ordena un array de numeros recibiendo un callback
let numbers = [1,2,45,12,54,23,65,98,18]
numbers2.sort((a,b)=>a-b) -> [1, 2, 12, 18, 23, 45, 54, 65, 98]

• Concatenar y transformar a string

• join

une las posiciones de un string y le agrega un separador en caso de usar ('') o (',')
let array = [1,2,3,4,5] array.join('-') -> 1-2-3-4-5

• concat

concatena 2 arrays
let a1 = [1,2,3] let a2 = [4,5,6] a1.concat(a2) -> 1,2,3,4,5,6

• Encontrar elementos

• indexOf

devuelve el index del argumento en el array
array = [1,2,3,4,5] array.indexOf(4) -> 3

• find(callback)

devuelve el valor que cumpla el callback
array = [1,2,3,4,5] array.fin(number => number > 2) -> 3

• findIndex(callback)

devuelve el index que cumpla el callback
array = [1,2,3,4,5] array.findIndex(number => number > 2) -> 2

• Eliminar elementos (spread operator)

• set

eliminar elementos duplicados y retornar el array resultante con (spread operator ...)
array = [1,2,3,4,2,5] [...new Set(array)] -> 1,2,3,4,5

• function removeDuplicates

almacenar el set en una funcion
let array = [1,2,3,4,4,5]
const removeDuplicates = array => [...new Set(array)]
removeDuplicates(array) -> 1,2,3,4,5

• Math.min()

retorna el valor mas bajo de un array (solo usando spread operator)
let array = [1,2,3,4,5]
Math.min(...array) -> 1

• Math.max()

retorna el valor mas alto de un array (solo usando spread operator)
let array = [1,2,3,4,5]
Math.max(...array) -> 5

• Recorrer arrays

• for of

muy parecido al for each de JAVA es utlizado para recorrer arreglos utilizando un alias como elemento

let arr = [1,2,3,4,5]
for(element of arr){
console.log('elemento numero: ' element)
} -> elemento numero: 1 ... 5

• forEach (callback)

recorre un array haciendo un callback que recibe dos argumentos: element, index arr.forEach(el,i)

teachers = ['javier','jorge','camilo','juan']
teachers.forEach((el,i) =>{ console.log((i+1) + '. ' + el) })

• metodos .some() y .every()

recorre un array, recibe un callback y retorna true si algun elemento cumple la condicion (some) o retorna true en caso de que todas la cumplan (every)

let numbers = [1,2,3,4,5,6]
// existe un elemento mayor a 5
numbers.some(el => el > 5) -> true
//todos son mayor a 2
numbers.every(el => el > 2) -> false
})

• Metodos map() filter() reduce() [no odifican el array original]

• .map(callback)

transforma todos los elementos del array

let array = [1,2,3,4,5]
array.map(el => el*el) -> [1, 4, 9, 16, 25]

• .filter(callback)

filtra los elementos que cumplan con la condicion

let array = [1,2,3,4,5]
array.filter(el => el > 2) -> [3,4,5]

• .reduce(callback)

reduce todos los elementos a un unico valor

let array = [1,2,3,4,5]
array.reduce( (a,b) => a + b) -> 15
